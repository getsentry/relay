Struct Deserializer

Source 
pub struct Deserializer<'a, D, T> { /* private fields */ }
Expand description

A Deserializer that maps all values through a Transform.

This deserializer wraps another deserializer. Values are transformed inline during deserialization and passed directly to the Deserialize implementation. All errors are passed through without modification.

§Lifetime

The lifetime parameter is an implementation detail. new returns a transforming deserializer with static lifetime.

§Example

ⓘ
struct Identity;

let json = "42";
let json_deserializer = &mut serde_json::Deserializer::from_str(&json);
let deserializer = Deserializer::new(json_deserializer, Identity);

let number = deserializer.deserialize_u32(deserializer).unwrap();
assert_eq!(number, 42);
