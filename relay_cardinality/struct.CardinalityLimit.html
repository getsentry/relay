A cardinality limit.

Fieldsยง

ยงid: String

Unique identifier of the cardinality limit.

ยงpassive: bool

Whether this is a passive limit.

Passive limits are tracked separately to normal limits and are not enforced, but still evaluated.

ยงreport: bool

If true additional reporting of cardinality is enabled.

The cardinality limiter will keep track of every tracked limit and record the current cardinality. The reported data is not per limit but per scope. For example if Self::scope is set to CardinalityScope::Name, the current cardinality for each metric name is reported.

ยงwindow: SlidingWindow

The sliding window to enforce the cardinality limits in.

The cardinality limit.

Scope which the limit applies to.

Metric namespace the limit applies to.

No namespace means this specific limit is enforced across all namespaces.
